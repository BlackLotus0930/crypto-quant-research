## 0. 目标

### 0.1 两个板块：模型 → 交易系统（分开做，按顺序）—— 仍成立

- **最终目标**：自动交易系统赚钱。但赚钱是**整条栈**(模型 × 组合 × 执行 × 杠杆)的结果，不是模型一个人的指标。
- **两块**：① **模型(alpha 研究层)** `data→distribution`，找真实且新颖的横截面 alpha；② **交易系统(组合+执行+杠杆)** sizing/换手/maker/容量/杠杆。
- **为什么分开**：真实量化就这么分；混在一起会拿烂交易系统的指标误杀好模型(E9 教训)。
- **但有个 2026-06 的重大修正**：原以为"模型毕业后再碰交易系统"。**实测发现:交易系统那一半(执行/适应/宽度/杠杆)才是把"薄而真的 alpha"变成钱的真杠杆**——它不是模型的下游附属，是与模型并重的主战场(见 §1.3)。

### 0.2 模型毕业标准（三条全过）—— 仍成立，加第 0 条

> 全部 holdout、样本外、跨 regime。

0. **【新增·铁律】数据无幸存者偏差 + 无前视 + 不偷看**：universe 必须 point-in-time(含已退市标的)；执行参数在 val 选、holdout 只报；归一/窗口/标签全因果。**任一不满足，下面三条的数字都不算数**(教训：E16 净@5 +1.22 被幸存者偏差灌高→作废重做，见实验.md E16/E18)。
1. **真实**：holdout IC 的 **t 值 ≥ 3** 且 **val→holdout 保留 ≥ 50%**(不塌)。
2. **新颖**：剔全套线性因子(反转/动量/波动/量)后**残差仍预测**——novelty 占比 ≥ 50% 且残差 t ≥ 3。**这是 thesis 唯一直接的检验。**
3. **可交易(gate)**：成本前沿某执行档扣现实成本(加密 taker ~5bps)**净 Sharpe ≥ 0.5**。**且必须逐年为正、含最新 regime**(教训：LSR 聚合正但 2026 转负=假，见实验.md E17)。

## 1. 认识论与方法论（本版核心重写）

### 1.1 Software 2.0 找 alpha —— 【已验】保留

- 市场复杂度远超人手工建模上限；拒绝控制论、拒绝人类二手因子、拒绝"真实价值"锚点。
- 人给数据和目标，神经网自己学规则；alpha 藏在人描述不出的高阶联合结构里。
- **已验(E10)**：剔掉整套线性因子后，模型残差**仍预测**(t~8、占比>50%)。**模型确实挖到了高阶结构——认识论这一半成立。** 这是我们 ≠「又一家用 ML 的量化」的分界。

### 1.2 为什么"加更多/更广数据"在市场里没用 —— 【已证伪 想法.md 的数据扩张框架】

> 想法.md 押"K 条正交流 → 两两 K²、高阶 2^K → 乘性更多 alpha"。**E11-E15 实测：0-for-4-然后-微正**(订单流冗余/funding 无信号/跨市场有害/联合 OI+多空比+资金费仅 +12% 且 NO-GO)。**这个框架错了。为什么——市场与正常 AI 的三个根本差异：**

| | 正常 AI(视觉/NLP) | 市场 |
|---|---|---|
| 信噪比 | 高(输入→标签是强函数) | **极低(~99.7% 噪音)** → 多维度只拟合噪音=过拟合，不是更多信号 |
| 平稳性 | 静态 | **非平稳，关系会漂** → "更近"胜"更多"，旧数据是过期 regime |
| 对抗性 | 非对抗 | **对抗/有效(Grossman-Stiglitz)** → 可榨 alpha 被**竞争**封顶，不被你的数据/模型封顶 |

**推论：在市场里，"数据×规模"不是杠杆**(那是高信噪、静态、非对抗世界的直觉)。再多正交数据也突破不了"竞争把 alpha 钉在成本线 + 关系一直漂"这两条。**数据扩张到此封存**(留作"偶有微正"的边角，不是主线)。

### 1.3 真正的杠杆 = 方法 —— 【实验跑出来的新主线】

bottleneck 从"有没有 alpha"(有，且真且新)变成"**怎么把薄而真的 alpha 低成本、宽、带杠杆地榨出来**"。三个真杠杆：

1. **适应(walk-forward / 在线重训)治非平稳**【已验 E16】：每隔一段用最新数据重训、预测下一段。静态模型 val→holdout 崩(E9)、LSR 2026 转负(E17)；而 walk-forward 让 IC 跨 5 个 regime(含 2026)全守在 ~0.016-0.018 不塌。**非平稳从"alpha 死了"变成"alpha 可追"——这是项目第一个硬突破。**
2. **执行治成本**【已验 E16】：信号有"慢成分"，EMA 平滑 + 低换手(~50/年)交易 → 成本几乎为零 → 薄 IC 落袋成净 Sharpe。**快成分被 taker 成本吃掉，慢成分耐成本。** 进一步：**maker** 把成本翻成返佣，能解锁现在扔掉的快层(毛 Sharpe@0bps ~1.3 在那摆着)——未测，是最大一块桌上的钱。
3. **变现机器把薄边际变收益(Medallion 公式，非神预测器)**【待做】：**IR ≈ IC × √(独立信号数)**。我们 IC~0.016 和顶级量化一个量级；他们赢在 **宽度(很多近独立弱信号)× maker 执行 × 杠杆**，不在更深的预测器。
   - **多信号宽度**：从原始数据造 4-6 个机制不同的弱信号(反转/季节/持仓/carry/vol择时)再组合 → ρ 封顶 ~1.3-1.8× + 抗衰减(它们不同步失效)。
   - **杠杆**：干净 Sharpe × 3-5 = 把 Sharpe 变成年化%(你要的"高年化"= Sharpe × 杠杆，不是更高准确率)。

### 1.4 残酷的尺子 = 一等公民 —— 【血泪】

低信噪 + 对抗 → **最容易骗自己**。每个看着好的结果都要主动推翻。已踩并固化为门(§0.2 第 0 条)：
- **假 GO 撞 seed**(E2)、**LR 退火 bug**(E10)、**assign_split 重叠 bug**(E16,WF 每块评估到数据末尾、book 重叠 3×)、**幸存者偏差**(E16,universe 只含活到今天的币→反转被灌高)、**holdout 偷看**(成本前沿全局最优=乐观上界)、**研究者过拟合**(同一 holdout 看几十遍→唯一真解=向前 paper trade 不可偷看的未来)。

## 2. 数据

### 2.1 来源【加密为主】
- **加密(主)**：Binance Data Vision **免费** flat files——永续 1h/1m K线、fundingRate、metrics(OI/多空比)、premiumIndex(基差)。镜像 `data/raw/binance/`。
- **US 跨资产**：Massive 分钟(E7 沉没成本)，仅作对照——**跨市场喂模型已证有害(E13)，不进主线**。

### 2.2 频率【60min；不碰秒-分钟 HFT】
主 = 60min(可交易 band)。不往秒-分钟走(延迟/做市的 HFT 战场，无 edge；E12 实测快信号是价格反转的冗余影子)。

### 2.3 Universe【point-in-time，含死币 —— 本版重大修正】
- **必须 point-in-time**：从 Vision 归档枚举**所有曾存在**的 USDT 永续(含已退市，`download_binance --all`，732 个 vs 当前 200 幸存者)，每根 bar 取**当时**最流动 top-N(死币流动时进、退市后退) + 时长门槛控 N²显存(`build_panel_pit`/`build_tensor_pit`，N≈272)。
- **为什么**：原来 `top_symbols` 只取 status==TRADING = 幸存者偏差，把反转策略灌高(E16 审计)。**这是 §0.2 第 0 条的硬要求。**

### 2.4 数据管道
- `build_crypto`/`build_panel_pit`(K线→bar) + `build_*_tensor`(per-coin 因果归一 + tod)。
- 原则：机械卫生 + scale-free 因果归一 + 零人类观点 + 全程因果。守卫 `check_invariants.py`(因果归一 maxdiff=0)。

### 2.5 数据获取漏斗 —— 【降级：数据不是主线】
- 探针只筛**正交性(corr)**，**不能替我们找 alpha**(高阶对线性探针不可见；"探不到"才可能还在)。
- **但既然数据扩张已证伪(§1.2)，这条漏斗降为低优先**。新数据流要进，先问"它能不能进**新的弱信号**(宽度，§1.3)"，而不是"喂进同一个模型加通道"(已证基本没用)。
- 工具留用：`probe_orthogonal.py`、`vol_probe.py`、`monetize_probe.py`。

## 3. 预测任务【基本不变】
- 输入：全 universe context window + Δt；输出：每标的 Δt 后收益的 K 维分布；一次前向预测当天所有标的。
- Target：**绝对 log return、尺度归一空间**(除因果慢尺度，serving 乘回)，不减基准(减基准=人类先验)。alpha 主场在横截面相对维度，模型经资产轴 attention 自学共同模态。
- 输入只用原始 OHLCV，无人造衍生量(允许机械卫生 + 因果归一)。

---

## 4. 模型架构【基本不变，它能工作】
- **约束**：资产置换等变 + 无 ticker ID(=泛化 + 数据倍增器) / mask 抗 ragged / 因果训练=推理 / scale-free 内部一致 / 可放大 N。
- **编码器(轴向)**：时间轴 per-asset causal self-attn(权重跨资产共享) + 资产轴 masked 置换等变横截面混合(把"同时发生的联合结构"注入)。N~数百用 full attn(O(N²))；放大用 latent 瓶颈(M 个 latent = 模型自学的高阶共同模态)。堆 2-6 层。
- **分布 head**：anchor token → FiLM 注入 dt → 单调分位头 → K 分位。
- **损失**：横截面 IC(Pearson)为主 + λ·CRPS 辅(纯 CRPS 对方向信号失明，E0 证伪)。
- **规模/正则(小数据纪律)**：等变权重共享已是强正则 + dropout/weight decay/早停。**注意(§1.2):低信噪下加容量=过拟合，模型不是越大越好**——按"加容量样本外是否还涨"实测，多半不涨。
- **N² 显存铁律**：每 epoch 计算 ≈ 样本数 × N² × epochs，**与 bs 无关**(bs 翻倍步数减半、每步翻倍，抵消)。提速靠**减 N / 减锚点 / 减 epoch / 滚动窗**，不是加 bs(E18 踩过)。

## 5. 评估（经济口径）【已验，加 §1.4 的反自欺】
1. **模型价值(主判据)= 真实 × 新颖**：`eval/report.py` 出 IC t值 + val→holdout 保留 + 剔全因子残差(novelty)。`run_gonogo` VERDICT 据此判。
2. **可交易 gate(非 verdict)**：固定不学习策略翻成仓位 →
   - **sizing = 风险平价**(仓位 ∝ z(信号)/慢波动；旧 conviction=信号/spread 毁账户，E3)。
   - **慢执行 = 真杠杆**(`cost_aware.py`：EMA 平滑 + 低换手；§1.3)。
   - **成本**：taker ~5bps / maker ~2bps；报 2/5/10bps + 调仓周期前沿。
   - **gate = 某执行档净 Sharpe ≥ 0.5 且逐年(含最新)为正**。
3. **【新增·必做】对抗审计**(`audit.py`)：① 年化不虚(per-bar Sharpe ≈ 日块 Sharpe、pnl 自相关低) ② 无前视(加 1bar 执行延迟仍正) ③ 符号打乱≈0 ④ 幸存者(universe 有退市) ⑤ 参数 val 选不偷看。**过审计才信数。**

## 7. 路线图（实验史 + 当前）

- **v0-v2(legacy)**：akshare 日频 → US 分钟。信号真但薄、被成本绑。
- **v3 加密(E8-E15)**：转加密(E8)；真模型真+新但 taker 不可交易(E9)；**eval 重构+修 LR bug(E10)**；**数据扩张 0-for-5(E11 订单流/E12 漏斗/E13 跨市场有害/E14 残差化/E15 联合+12%NO-GO)→ 数据不是杠杆(§1.2)**。
- **v4 方法转向(E16-E18，当前)**：
  - **E16 walk-forward 适应** = 第一个硬突破:消除非平稳 IC 崩塌(5 块含 2026 不塌)。但审计揪出 **assign_split 重叠 bug(已修)** + **幸存者偏差** → 净@5 +1.22 不可信。
  - **E17 plan B 风险/波动探针**:可预测但大半机械;LSR 持仓反向真但非平稳。
  - **E18(进行中)**：point-in-time 含死币重跑(N=272)，cost_aware/peryear/audit。**带死币后 net@5 仍 >0.5 且 2026 正 = 项目首个过全部怀疑的可交易结果。**
- **下一步(按 §1.3 优先级)**：① pit 定生死 ② maker 执行(解锁快层，最大桌上钱) ③ 多信号宽度(IR×√N) ④ 杠杆。**不再往模型里堆数据。**

### 当前诚实状态
- **机制已验证**：真+新颖 alpha 存在(E10)、非平稳可追(E16 适应)、慢成分耐成本(E16 执行)、尺子可信(E16 审计套件)。
- **可交易幅度待定**：E16 的 +1.22 被幸存者偏差灌高，pit(E18)才是真数。
- **一句话**：方案的**机制对了、第一次跑出干净的正**;但"能赚钱、能赚多少"要等 pit + maker + 宽度 + 杠杆。**赢在方法,不在数据。**
