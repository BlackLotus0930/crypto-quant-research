# 核心判断

现实是一个高维的联合结构，维度高到人类无法直接理解。科学几千年做的事，是把这个结构强行压缩成低维的方程——找少数几个变量之间的线性因果关系，写下来，传递给别人。这个方法在现实碰巧是低维的地方成功了，比如物理。但金融不是这种现实。金融是不确定有多少维的非平稳系统。

经济学和金融学建立的所有概念——Beta、相关系数、利率传导机制——都是两两关系，是人类认知能处理的上限，是对真实结构的低维投影。这些投影被发现之后就被套利干净了，剩下的 alpha 藏在高阶的联合结构里，藏在变量之间同时发生的模式里，是人类用语言和方程永远无法触碰的部分。

大规模数据第一次让直接在高维里找结构变得可能：不需要人先猜测，不需要先降维，让模型从原始数据里还原出人类从来没有描述过的结构。这是金融建模的一次根本性的方法转变——不是更好的因子模型，而是完全不同的认识论。

## 核心判断的精炼（2026-06，跑过 E0–E7 + 市场快筛之后）

把空想落到数据上之后，几条要钉住：

**1. 目标和所有量化一样，差异只在「怎么找」。** 目标永远是：短期绝大多数是噪音（单票日收益 >99.7% 不可预测），在噪音里捞那一丝可预测的横截面 alpha。我们没有、也不可能重新发明这个目标。真正不同的**不是「用了模型」**（每家量化都用 ML），而是 **「不预设因子、不先降维、喂最生的数据，赌 alpha 在人描述不出的高阶联合结构里」**。信号从一条人写的公式，变成藏在权重里、手写不出来的隐式表示。

**2. 这个赌注目前「未被证明」。** 诚实说：到 E7 为止，我们这套无先验模型的 holdout IC ≈ 0.013，和手搓线性反转因子（≈0.016）**一个量级**——我们还没演示出「高阶结构里有更多、且我们抓得到」。整个后续（scale、换市场、更生数据）存在的意义就是逼出那个「高于简单因子基线」的边缘。**逼不出来，thesis 就是错的，我们只是昂贵地复刻线性因子。**

**3. Grossman-Stiglitz：alpha 恰好停在「成本线」上。** 市场有效到「剩下的 alpha 刚好够补偿挖+交易它的成本」。E0–E7 在美股大盘看到的「IC 真但扣成本勉强转正」**正是这个均衡的样子**，不是失败。推论：**成本线的高度取决于「谁在跟你抢」**——所以真正的杠杆是 **「去竞争更弱的场子」**，而不是在被 Citadel 盯死的美股上比人聪明。

**4. 市场选择 > 模型技巧（市场快筛实测，见 实验台账.md E8）。** 同一把廉价尺子量股票 vs 加密：去弹跳的耐用反转（rev5_skip）**加密尾部 +0.029 IR1.7 vs 股票小盘 +0.007 IR0.6**——**加密低效 2–4 倍、稳 2–3 倍**。两市场同源（都反转主导、越不流动越低效），但加密整条曲线抬高一截，且成本更低（永续 ~5bps vs 小盘股 15–30bps）。→ **战略转向加密**。

**5. 「换数据种类」不是扩张，是 thesis 的推论。** OHLCV 本身就是一个人类先验（人挑出来留下的有损摘要）。要找人描述不出的结构，却喂人预先消化过的数据，自相矛盾。**数据才是杠杆，且是「更生 + 更宽 + 更异质，再联合编码」**——人描述不出的结构最可能藏在多条异质数据流的「同时发生」里，而 any-variate 架构正是为吃这个造的。加密恰好**免费给最生的数据**（逐笔/盘口/taker 买量/funding/链上）。

## 如何扩大潜力（数据扩张方法论，2026-06，有实证支撑）

**实证锚点（E9 输出解剖）**：加密模型的预测力，**剔掉整套标准线性因子（反转/动量/波动/量）后，残差仍有 IC +0.0082、IR 4.9**——一半以上不是任何明显因子。即**模型在单一 OHLCV 流里已经挖出了「人描述不出」的结构**（跨币联合结构）。机制证实：模型真能挖联合结构。

**第一性原理**：对无先验模型，**alpha 活在「喂进去的数据流之间的联合结构」里**。模型只能找到数据里存在的关系。
- 给一种数据 → 只能挖那种数据内部 + 其横截面的联合结构（有限、且被竞争）。
- **潜力被「在场的数据流」封顶。** 更多**正交**的流 = 更多联合结构 = 乘性放大的潜力（K 条流，两两 ~K²、高阶 ~2^K；人只描述得了少数明显的两两关系，模型挖全部）。

**关键限定词 = 正交**：新流的价值 = 它带来的、现有流没有的、且与现有流有联合结构的信息。加冗余流（又一个价格衍生特征）≈ 没用；加正交流（订单流、链上、跨市场）价值大——因为价格不含它们，其交互无人探过。

**正交数据轴（按潜力排）**：① 微观结构/订单流（taker 失衡、盘口、逐笔；正交于 OHLCV、最少人探、加密免费）② 跨市场/跨资产（加密+股+FX+利率+商品 当联合变量；跨市场耦合最丰、最不可解）③ 链上（加密独有、与价格正交）。④ 更多资产（齐涨=冗余，低）⑤ 更细频率（放大器，不加正交信息）。

**硬约束（否则翻车）**：更多流 = 更多维度。低信噪比 + 有限样本下，维度多 → **过拟合**，不是更多 alpha。**扩张必须是 (数据多样性 × 样本量 × 模型规模) 的联合放大**——不能把多种数据砸到小模型/少样本上。

> **一句话**：潜力 = 正交数据流的联合结构空间。放大潜力 = 加正交流（订单流→跨市场→链上）+ 联合编码 + 同步扩样本与模型规模。瓶颈已从「有没有结构」（E9 证明有）变成「喂够多样数据 + 够大模型，把结构挖厚到过成本」。

## 灵感来源

### 传统自动驾驶的假设

- 传统自动驾驶需要：摄像头 + LiDAR + 雷达 + 超声波 + 高精地图 + GPS
- 他们的假设：更多的信息来源 = 更多的冗余，可以互相弥补

### Software 2.0

Karpathy 在 2017 年写过一篇影响深远的文章，叫 **Software 2.0**。


|     | Software 1.0 | Software 2.0          |
| --- | ------------ | --------------------- |
| 范式  | 人写规则 → 计算机执行 | 人提供数据和目标 → 神经网络自己学出规则 |


这个信念应用到自动驾驶上，结论就很直接：

传统方案（LiDAR + 高精地图 + 手工感知算法）是 Software 1.0 思维——工程师试图把所有驾驶知识编码成规则。这条路在 2012 年之前的计算机视觉领域已经证明是死路。

### 为何特斯拉选择纯视觉 + 真实数据训练

```
部署当前模型到车队
    ↓
模型在某些场景下不确定、预测矛盾、或事后回溯发现错了
    ↓
自动触发：把这段数据片段上传
    ↓
标注这些困难样本
    ↓
加入训练集，重新训练
    ↓
部署更新的模型
    ↓
循环
```

这个循环的关键在于：困难样本是从真实驾驶中自动发现的，不是人为猜测的。

Waymo 工程师再聪明，也没办法预想到现实道路上所有的奇怪情况。但特斯拉的百万辆车会遇到所有情况，而且会自动把困难情况标记出来。

### 对「复杂度」的不同判断

他和传统工程师的分歧，本质是对「复杂度」的不同判断：

- **传统派认为**：开车是一个可以分解的问题。感知、预测、规划、控制，每层都可以用人类理解的规则和传感器来处理。
- **Karpathy 认为**：开车的真实复杂度远超人类能手工建模的上限。

## 量化交易

现在的量化交易基本是控制论的思路：

```
测量现状 → 和目标比较 → 算出误差 → 输出纠正指令 → 循环
```

### 我的判断

交易不需要有人类分析的知识。已有的知识对于 AI 来说可能是噪音。

### 有标准 vs 没有标准

- **有标准**：会优化这个标准，然后其他地方炸裂。
- **没有标准**：不按标准行事，并不意味着没有标准。我以为我放弃了标准，其实只是把标准从显性变成了隐性。

## 端到端建模

所有的金融概念、术语、公式都是人类为了自己的预测而创造出来的。「真实价值」是一个人造概念，是人类试图在价格之外再造一个锚点，但这个锚点本身也是人估算出来的。价格就是价格，它已经包含了所有人对所有信息的反应。

**端到端模型的目标：预测下一个状态。**

### 数据

- **交易目标（演进）**：原定美股多资产联合；**2026-06 转向加密**（永续合约）——市场快筛实测加密低效 2–4 倍、成本更低、24/7 无隔夜、原生可做空、且**免费给最生的数据**（逐笔/盘口/taker/funding）。美股仍是对照基线。
- **Universe**：股票侧=全部流动性美股 + 跨资产 ETF（作上下文）；加密侧=Binance USDT 永续 top ~200（按成交额）。
- **训练 = 推理同源**
- **数据源演进**：日频曾用免费 akshare/新浪（E0–E6，但 akshare mask 仅 0.298、70% 空，是混淆变量）；切付费 **Massive.com** 拿美股分钟（E7，频率没产生质变、被成本绑）；**现转 Binance 免费加密数据**（最生、最低效）。详见 [数据.md](数据.md)、[设计.md](设计.md)、[实验台账.md](实验台账.md)。

### 推理

- **输入**：所有资产的 context window（市场当前状态）+ Δt
- **输出**：目标资产的 K 维分布

### 模型

- 单步、分布预测
- **损失**：原设想 CRPS；**实操已改为横截面 IC 主 + CRPS 辅**——CRPS 对"能赚钱的微弱方向信号近乎失明"（见 实验台账.md E0）。分布输出仍保留（供风险平价 sizing）。
- Joint representation
- **Baseline**：random walk（分布意义）+ 横截面 IC≈0
