## 4. 对账(跑 2-4 周后,这才是重点)

把前向实盘和 backtest 对:

| 指标 | backtest 预期 | 前向看 |
| --- | --- | --- |
| 折年化(unlevered) | ~+15%/yr | `cum_pnl/天数*365` |
| 分腿 | funding 腿为主、价格腿小负、借币小拖累 | log 的 fund_pnl ≫ \|price_pnl\| |
| 夏普 | ~4(偏乐观) | log 的 bar_pnl 算 |

**对得上 → 那 +15%/yr 是真的,可上小额实盘;对不上(尤其 funding 腿缩水/借不到券)→ 找出差在哪再说。** 这是整套东西"真不真"的最终答案。
